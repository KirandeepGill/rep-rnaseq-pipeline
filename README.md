# Reproducible RNA-seq Pipeline Using Docker

## Overview

This repository contains a reproducible bulk RNA-seq analysis pipeline implemented using Docker. The goal is to demonstrate bioinformatics workflow using publicly available sequencing data.

The pipeline analyzes a subset of six RNA-seq samples from the Gene Expression Omnibus (GEO) dataset **GSE236374**.

---

## Objectives

- Build a reproducible RNA-seq workflow
- Containerize the analysis environment using Docker
- Perform quality control, alignment, read counting, and differential expression analysis
- Document every step of the workflow for reproducibility
